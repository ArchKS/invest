# 分析面板行业的周期


这里的“面板行业”我按显示面板来分析。核心结论：**面板是强周期行业，但LCD大尺寸面板的周期正在从“产能扩张-价格暴跌”的大开大合，转向“供给纪律+尺寸升级+技术替代”的弱周期；OLED则仍处在渗透成长周期，但竞争和价格压力会越来越强。**

**1. 面板周期怎么形成**
面板周期主要由三层叠加：

1. **产能周期**  
   面板厂投资大、折旧重、产线建设周期长。高景气时扩产，等新产能投出来，需求可能已经降温，价格就容易崩。  
   LCD时代最典型：高世代线放量、稼动率上升、价格战、亏损、减产，然后再修复。

2. **库存/价格周期**  
   终端品牌看到需求好会提前备货，面板价格上涨；一旦电视、手机、笔电卖不动，下游砍单，面板库存累积，价格快速下跌。  
   2022年就是典型下行：TrendForce当时指出LCD TV面板价格跌至新低，供需失衡严重，厂商被迫减产。

3. **技术替代周期**  
   LCD、Mini LED背光、OLED、Micro LED不是同一条周期。  
   LCD偏成熟、看成本和供给纪律；OLED看渗透率、良率、价格下探；Micro LED更偏远期技术储备。

**2. 当前处在什么位置**
截至2026年5月看，行业不是全面上行，而是**分化修复**：

- **大尺寸LCD TV面板：偏底部修复/温和紧平衡。**  
  TrendForce称，2026年一季度因春节减产、以旧换新补贴、世界杯备货等因素，LCD TV面板供需转紧；同时长期看LCD TV面板缺乏新增大投资，对价格和产业健康度是正面信号。  
  价格上，集邦2026年4月20日数据里，55英寸UHD Open-Cell均价约126美元，环比持平，说明不是暴涨，而是稳价。

- **手机面板：偏弱。**  
  TrendForce预计2026年全球手机面板出货约21.4亿片，同比下降7.3%。AMOLED占比继续提升，但手机需求疲软、存储器涨价挤压整机成本，会让品牌对面板继续压价。LCD手机面板压力最大。

- **OLED显示器/IT OLED：成长周期。**  
  TrendForce称2025年全球OLED显示器出货273.5万台，同比增92%，预计2026年再增51%。Omdia也预计2026年平板OLED面板出货同比增长39%。这说明OLED在显示器、平板、笔电等IT场景进入较快渗透阶段。

- **LCD产能结构：向中国大陆和高世代线集中。**  
  TrendForce预计2026年8.6代线占全球LCD产能约26%，小世代线继续退出。TCL收购LGD广州LCD厂后，LCD TV面板集中度进一步提高。集中度提升会削弱过去那种恶性扩产周期，但不会消灭周期。

**3. 未来周期的关键变量**
最值得盯这几个指标：

- **面板价格：** TV 55/65/75英寸、显示器、笔电面板报价是否连续上涨。
- **稼动率：** 80%以下通常说明需求弱；接近90%且库存低，价格更容易涨。
- **下游库存：** 电视、手机、PC品牌是否补库存。
- **大尺寸化：** 电视出货台数可能不强，但平均尺寸变大，会拉动面积需求。
- **OLED渗透率：** 手机AMOLED、IT OLED、车载OLED的渗透速度。
- **新增产能：** LCD看高世代线和旧线退出；OLED看8.6代线、印刷OLED、良率爬坡。
- **宏观和补贴：** 中国以旧换新、体育赛事备货、美国关税、存储器涨价都会影响终端需求。

**4. 一句话判断**
面板行业现在不是2020-2021那种全面大牛周期，更像是：**LCD大尺寸进入供给纪律后的弱复苏周期，手机LCD继续下行，OLED处于结构性成长但利润率会被竞争稀释的周期。**

# 历史周期复盘

可以。下面按**大尺寸LCD面板主周期**复盘，OLED单独看，因为OLED更多是渗透率周期，不完全跟LCD价格周期同步。

我把一轮周期定义为：**价格底部 → 补库存/需求上行 → 产能或稼动率上升 → 价格见顶 → 库存累积/砍单 → 减产出清**。

| 周期 | 阶段 | 上行原因 | 下行原因 | 复盘结论 |
|---|---|---|---|---|
| 2004-2007 | LCD TV导入期，5-7代线周期 | LCD替代CRT，电视/显示器需求扩张 | 2005新产能集中投产，2006明显过剩 | 第一轮典型“新技术高成长也挡不住产能过剩”。2006价格大跌，2007减产后价格反弹 |
| 2008-2010 | 金融危机周期 | 奥运、数字电视转换、LCD TV渗透 | 2008金融危机，需求骤降，厂商Q4大幅减产 | 2009靠中国刺激、价格下跌刺激需求反弹；2010又因品牌备货过度形成库存压力 |
| 2011-2012 | 后刺激去库存周期 | 2009-2010需求透支，厂商仍扩产 | 欧洲债务危机、美国疲弱、中国增速放缓、TV库存高 | 2011行业深亏，2012靠减产和需求修复企稳。这轮说明“终端库存”比面板厂出货更重要 |
| 2013-2015/1H16 | 中国8.5代线扩张周期 | 中国大陆产能崛起，BOE、华星等提升份额 | 新产能投放快于终端需求，2015库存上升、价格跌至盈亏平衡附近 | 这是LCD产业重心从韩台向中国大陆转移的周期，代价是全球价格中枢下移 |
| 2H16-2017 | 供给收缩小牛市 | 韩台厂关闭旧产线、转向大尺寸/OLED，32寸等规格短缺 | 不是需求暴增，而是供给收缩 | 这轮很关键：面板第一次体现出“供给侧收缩”带来的盈利弹性 |
| 2018-2019 | 10.5代线冲击周期 | 大尺寸化继续推进 | BOE、CSOT等高世代线放量，65/75寸供给冲击，贸易摩擦拖累需求 | 2019很多TV面板跌破现金成本，韩厂加速退出LCD，LCD行业进入“出清换格局”阶段 |
| 2020-1H21 | 疫情超级周期 | 宅经济拉动TV、笔电、显示器需求；驱动IC、玻璃、偏光片短缺；库存低 | 价格涨太多反噬终端需求 | 这是近二十年最强上行之一。DSCC统计2020年5月至2021年6/7月LCD TV面板均价涨幅约106%，32寸涨幅更极端 |
| 2H21-2022 | 超级周期崩塌 | 初期仍有惯性订单 | 疫情红利消退、通胀、加息、俄乌冲突、品牌库存高、需求断崖式转弱 | 2022价格跌至历史低位，厂商被迫大规模减产。这轮证明：面板涨价越猛，后面去库存越狠 |
| 2023-2024 | 供给纪律修复周期 | 2022大减产后库存下降，厂商按需生产，TV面板价格自2023年春季修复 | 终端需求并不强，价格上涨后品牌利润承压 | 这不是需求大牛市，而是“减产造出来的修复”。行业开始从拼份额转向拼利润 |
| 2025-2026 | 分化周期，还没完全闭合 | LCD TV缺乏新增大投资，厂商控产；大尺寸化提升面积需求 | TV/手机/笔电终端需求偏弱，存储器涨价挤压整机成本 | 当前更像弱复苏后的震荡期。LCD TV供给格局变好，但手机LCD、部分IT LCD压力仍大；OLED显示器、平板、笔电是结构成长线 |

**最重要的变化**

过去面板周期的核心是：**谁扩产，谁把行业带进下行。**  
现在LCD的核心变成：**谁控产，谁维持价格。**

这就是为什么2023之后的周期和以前不一样。以前LCD每轮高景气后都会有新产线冲出来；现在LCD TV大规模新增投资减少，韩厂退出，中国大陆龙头集中度提高，价格周期可能变得没以前那么暴烈，但仍然会跟着终端库存波动。

**每轮周期的共同规律**

1. 面板价格最差时，往往不是行业最危险的时候，反而是出清开始的时候。  
2. 行业盈利最好的时候，通常已经接近周期后半段。  
3. 判断拐点看三件事：价格是否止跌、稼动率是否回升、下游库存是否回到健康水平。  
4. 判断下行看三件事：终端销量走弱、品牌砍单、产能/稼动率还在高位。  
5. LCD看供给纪律，OLED看渗透率和良率，Micro LED暂时还不是主周期。

**当前判断**

截至2026年5月，LCD TV面板处在“供给纪律支撑下的弱周期”，不是2020-2021那种超级上行；手机面板偏弱；OLED显示器、平板、笔电仍是成长周期。面板行业还会周期波动，但LCD的周期弹性正在变小，OLED/高端IT面板的结构弹性正在变大。

# 未来新增产能



按已公告/在建项目粗算，**未来三年真正新增的面板产能主要是8.6代OLED，名义产能约10万片玻璃基板/月**，折合约**61万平方米/月、730万平方米/年**。LCD TV大尺寸新增不多，更多是既有线爬坡和旧线退出。

**OLED新增产能**
| 厂商 | 产线 | 预计量产 | 公开/测算产能 |
|---|---|---:|---:|
| Samsung Display | A6 8.6代OLED | 2026 | 约1.5万片/月 |
| 京东方 BOE | 成都B16 8.6代OLED | 2026 | 3.2万片/月 |
| 维信诺/合肥国显 | 8.6代AMOLED | 2027 | 3.2万片/月 |
| TCL华星 | 广州t8 8.6代印刷OLED | 2027-2028 | 2.25万片/月 |
| **合计** |  |  | **约10.15万片/月** |

这里的核心是IT OLED，也就是笔电、平板、显示器，不是电视OLED。由于8.6代基板面积约6平方米，满产后对应约**60万平方米/月以上**的OLED基板投入能力。

**LCD新增产能**
LCD没有类似2020年前后那种大规模TV面板扩产。比较明确的是：

- 天马厦门TM19 8.6代LCD：设计月产能约**12万片/月**，已有报道提到2027年初可能扩至**16万片/月**，主要面向IT、车载、工业显示。
- BOE B19、TCL华星T9等更多是制程优化、产能爬坡，不是全新大投资。
- TrendForce预计2026年8.6代线占全球LCD产能约**26%**，但同时台厂小世代线退出，5代及以下LCD产能占比降至约**4.7%**。

所以结论很直接：**未来三年LCD净新增压力有限，OLED新增压力较大。**  
如果只看OLED，新增约**10万片/月8.6代基板**；如果把LCD爬坡也算进去，天马TM19这类IT LCD新增/爬坡产能也有十万片/月级别，但会被旧线退出部分抵消。
