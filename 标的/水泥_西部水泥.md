截至 **2026-04-30**，我按“水泥产能，含粉磨能力”来口径化。

**西部水泥**
西部水泥海外已经是核心增长区。2025年年报口径，海外已投产/已并表产能约 **1480万吨/年**：

| 国家/区域 | 产能 |
|---|---:|
| 埃塞俄比亚 | 630万吨 |
| 乌兹别克斯坦 | 250万吨 |
| 莫桑比克 | 200万吨 |
| 刚果（金），含2025年12月收购CILU | 270万吨 |
| 卢旺达粉磨 | 100万吨 |
| 坦桑尼亚粉磨 | 30万吨 |
| **合计** | **1480万吨** |

2026年已有明确新增：乌干达 Moroto 熟料线及 Jinja 粉磨系统，合计约 **300万吨/年水泥产能**，Moroto 一期已于2026年4月竣工投产。因此当前运行/近运行海外产能可看作 **约1780万吨/年**。

未来三年确定性节奏：

| 年份 | 确定性新增 | 年末海外总产能，基准口径 |
|---|---:|---:|
| 2025A | CILU并表后为基数 | 约1480万吨 |
| 2026E | 乌干达约300万吨 | 约1780万吨 |
| 2027E | 莫桑比克Nampula约150万吨 | 约1930万吨 |
| 2028E | 暂无已披露明确投产项目 | 约1930万吨 |

还有一个重要弹性：西部水泥2026年4月公告拟收购南非 **AfriSam**，其水泥产能约 **450万吨/年**。这不是新建，是并购，且仍有交割条件。如果2026年完成，西部水泥海外产能会直接从约1780万吨升至 **约2230万吨**，2027年Nampula再投产后可到 **约2380万吨**。

**海螺水泥**
海螺有海外产能，而且不是刚开始出海。其海外布局主要在 **印度尼西亚、柬埔寨、缅甸、老挝、乌兹别克斯坦**。2025年柬埔寨金边海螺投产后，公开统计的海外熟料产能约 **1646万吨/年**；海外水泥产能此前披露约 **2295万吨/年**，2025年金边项目投产后应高于该数，但公司2025年报没有单独披露最新海外水泥产能明细。

海螺也有继续出海计划。2025年报明确说水泥主业要“国内国际两端发力”，积极开拓海外目标市场；2026年资本开支计划 **118.2亿元**。此外，2026年3月公开信息显示海螺有意在 **坦桑尼亚** 建设其非洲首个水泥厂。但和西部水泥不同，海螺目前没有披露类似“2026新增多少、2027新增多少”的海外产能排产表，所以不能可靠地逐年量化。

我的判断：**西部水泥未来三年的海外新增产能确定性更强、节奏更清晰；海螺已有海外产能且会继续出海，但短期新增产能披露不如西部水泥明确。**
