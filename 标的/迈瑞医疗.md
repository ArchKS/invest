## 分析迈瑞医疗的基本面

以下按“基本面”看迈瑞医疗（300760.SZ），数据以目前可查的最新正式披露为主：2025年年报，2026年一季报截至今天 2026-04-26 尚未披露。

**一句话结论**

迈瑞医疗仍是中国医疗器械里少数具备全球化、平台化、强研发和高现金流能力的公司，但它已经从过去的“稳定高增长白马”进入“国内承压、海外和新兴业务接棒、估值重定价”的阶段。现在看基本面，关键不是它是不是好公司，而是 2026-2027 年利润能否重新恢复增长。 

**核心财务表现**

2025年公司收入和利润出现明显下滑：

| 指标         |     2025年 |         同比 |
| ------------ | ---------: | -----------: |
| 营业收入     | 332.82亿元 |       -9.38% |
| 归母净利润   |  81.36亿元 |      -30.28% |
| 净利润       |  84.51亿元 |      -28.01% |
| 经营性现金流 | 101.45亿元 |      -18.40% |
| 研发投入     |  39.29亿元 | 占收入11.80% |

这是公司上市以来较少见的“营收、利润双降”。利润降幅大于收入，说明压力不只是销量问题，还包括价格、产品结构、费用和行业政策的共同影响。

来源：[2025年报摘要](https://paper.cnstock.com/html/2026-03/31/content_2194339.htm)、[年报图解公告](https://vip.stock.finance.sina.com.cn/corp/view/vCB_AllBulletinDetail.php?CompanyCode=80092993&gather=1&id=12048846)

**业务结构**

迈瑞现在主要看四块：

1. **体外诊断 IVD** 
   2025年收入约122.41亿元，同比下降9.41%，占集团收入超过36%，已是最大业务。国内受DRG/DIP、集采、检验结果互认等影响，试剂量价都有压力；但公司认为自身在免疫、生化、凝血等领域国内份额仍约10%左右，目标是三年提升到20%。这块是“短期受政策压制，长期看国产替代和份额提升”。

2. **生命信息与支持** 
   2025年收入约98.37亿元，同比下降19.80%，国际收入占该产线约74%。这块包括监护仪、麻醉机、呼吸机、除颤仪等，是迈瑞传统强项，但受医院资本开支周期影响明显。

3. **医学影像** 
   2025年收入约57.17亿元，同比下18.02%。亮点是超高端超声产品上市第二年收入超7亿元，同比增长超过70%。这说明高端突破仍在推进，但整体板块还没摆脱行业需求疲弱。

4. **新兴业务** 
   2025年收入约53.78亿元，同比增长38.85%，占收入约16%。包括微创外科、微创介入、动物医疗等。这是当前最亮的增量，但体量还不足以完全抵消传统三大业务下滑。

来源：[2025年报摘要业务分项](https://paper.cnstock.com/html/2026-03/31/content_2194339.htm)

**最大亮点：海外业务变成主引擎**

2025年国际业务收入176.50亿元，同比增长7.40%，占总收入53%；国内业务156.32亿元，同比下降22.97%。这意味着迈瑞已经从“中国医疗设备龙头”进一步变成“全球化医疗器械平台公司”。

海外亮点包括：欧洲增长约17%，国际新兴业务增长近30%；公司披露2025年突破近90家国际重点战略客户，其中包括全球前100家医院中的17家。北美、欧洲、亚洲高端客户渗透在推进。

这点很重要：如果只看国内医疗反腐、集采、医院预算压力，迈瑞会显得很难受；但如果看海外长期渗透率，它还有成长空间。

来源：[2025年报摘要海外业务](https://paper.cnstock.com/html/2026-03/31/content_2194339.htm)、[证券时报报道](https://www.stcn.com/article/detail/3712930.html)

**主要压力**

第一，国内医疗设备采购仍弱。公司自己也说国内医院普遍面临经营压力，采购预算缩减，设备行业仍是弱复苏。

第二，IVD政策影响没有完全结束。集采、检验结果互认、医保控费会持续压缩价格和试剂用量，短期会影响利润率。

第三，利润率下台阶。2025年毛利率、净利率均明显下降，说明过去“高毛利、高净利”的稳定模型正在被重新检验。

第四，海外扩张也不是无风险。高息环境、本地货币贬值、地缘冲突、贸易政策、注册准入和本地化运营都会影响节奏。

**估值怎么看**

截至4月中旬附近，迈瑞股价大约在156-160元区间，市值约1900亿元，TTM市盈率约23倍，市净率约5倍左右。相较过去几年，估值已经明显压缩，不再是高高在上的“白马溢价”。

但便宜与否取决于利润恢复速度：

- 如果2026年利润只有小幅恢复，23倍左右不算特别便宜。
- 如果2027年能重新进入双位数增长，当前估值就有吸引力。
- 如果国内需求继续弱、IVD利润率继续下滑，那估值仍可能被压。

来源：[富途行情页](https://www.futunn.com/stock/300760-SZ/news/announcement/all)、[理杏仁估值数据](https://www.lixinger.com/equity/company/detail/sz/300760/300760/fundamental/valuation/pe-ttm)

**我的判断**

迈瑞医疗的基本面底色仍然强：产品矩阵宽、研发投入大、现金流好、全球渠道强、品牌和装机基础深。但2025年已经证明，它不是不受周期和政策影响的“永动机”。现在的核心变量是三件事：

1. 国内设备招投标能否恢复；
2. IVD在集采和控费下能否靠份额提升抵消价格压力；
3. 海外和新兴业务能否持续高增长。

偏长期看，它仍是中国医疗器械里质量最高的一批资产；偏中短期看，业绩修复还需要验证。更适合用“跟踪复苏拐点”的思路看，而不是简单按过去高增长白马来外推。



## 如何评价迈瑞医疗的增长主要靠外购，而非内生，增长质量不高的情况

这个评价**有一部分成立，但如果说“迈瑞增长主要靠外购、内生不行、质量不高”，我觉得偏武断**。更准确的说法是：迈瑞正在从过去以内生产品线扩张为主，转向“内生研发 + 并购补赛道 + 全球渠道放大”的平台型增长模式。这个模式质量高不高，取决于并购后能不能产生协同，而不是并购本身。

迈瑞确实越来越依赖外延并购来打开新曲线。典型例子有两个：2021年以约5.32亿欧元完成收购 HyTest，补强 IVD 上游原料；2024年以约66.5亿元收购惠泰医疗控制权，切入心血管介入、电生理等高值耗材赛道。2025年年报里，公司也把“新兴业务”单独列为第四大产线，收入约53.78亿元，同比增长38.85%，而这块业务明显受益于惠泰等并购整合。

但这不等于增长质量低。原因有三点：

第一，迈瑞买的不是“报表利润”，而是补短板和扩赛道。HyTest解决的是 IVD 原料能力，惠泰解决的是心血管介入和耗材平台，这些都和迈瑞现有能力有产业关联，不是乱买资产堆收入。

第二，迈瑞本身仍有很强的内生能力。2025年研发投入约39.29亿元，占收入11.8%，这个强度在国内器械公司里仍然很高。超声高端化、监护设备海外突破、IVD装机和试剂体系、微创外科产品矩阵，都不是单纯靠收购能买出来的。

第三，收购后的资产质量目前不差。惠泰医疗2025年实现收入25.84亿元，同比增长25.08%；归母净利润8.21亿元，同比增长21.91%；经营现金流也增长。它不是亏损标的，也不是靠讲故事撑估值的早期资产。

真正需要警惕的是另一件事：**外购增长会掩盖老业务的真实压力**。2025年迈瑞整体收入下降9.38%，归母净利润下降30.28%，其中生命信息与支持、医学影像、IVD三大传统业务都承压。如果新兴业务高增长主要来自并表，而传统主业继续下滑，那市场会质疑“有机增长不足”，估值也会受压。

所以我的评价是：

迈瑞的外延并购不是低质量增长的铁证，而是成熟龙头进入新阶段的合理选择。医疗器械平台公司想进入高值耗材、介入、上游原料等领域，完全靠自研会很慢，并购是正常路径。但市场现在质疑它也有道理，因为2025年传统主业下滑太明显，新业务增长还没完全证明可以独立扛起集团增长。

判断增长质量，后面重点看三件事：剔除惠泰并表后，新兴业务的有机增速；传统三大业务能否恢复正增长；并购资产是否能通过迈瑞渠道在海外放量。只要这三点兑现，外购反而是高质量平台化扩张；如果兑现不了，那才会变成“买增长”。



## 全球范围内的医疗器械公司，靠内生增长多，还是靠并购多

全球医疗器械公司，尤其是**成熟龙头**，通常是“内生增长 + 持续并购”双轮驱动；如果只看大型公司，**并购贡献往往非常大，甚至超过内生增长**。

麦肯锡曾统计全球前30大医疗科技公司，2011-2016年期间，**超过60%的增长来自并购，扣除剥离影响后仍如此**。这说明在全球医疗器械行业，龙头靠外购补产品线、补技术、补渠道，是常态，不是例外。来源：[McKinsey](https://www.mckinsey.com/industries/life-sciences/our-insights/delivering-growth-m-and-a-in-medical-technology)

原因很简单：医疗器械不像消费品，很多细分赛道壁垒很深，比如心血管介入、骨科、神经介入、机器人、诊断原料、影像核心部件。大公司完全靠自研进入新赛道，周期长、失败率高、临床和注册风险大。所以全球龙头常见路径是：

1. 自己做平台、渠道、制造、临床和注册能力；
2. 并购小而强的技术公司或产品公司；
3. 用自身全球销售网络放大标的收入；
4. 再通过整合降低成本、扩大适应症、做组合销售。

所以，**“靠并购增长”本身不能说明增长质量低**。全球医疗器械行业里，高质量龙头反而常常是优秀的并购整合者。

典型例子：

- **波士顿科学 Boston Scientific**：长期通过并购补强心血管、内镜、泌尿、神经调控等领域。但它不是只有外延，2025年公司收入同比增长19.9%，其中有机增长仍高达15.8%，说明并购和内生创新都很强。来源：[Boston Scientific 2025 results](https://news.bostonscientific.com/2026-02-04-Boston-Scientific-announces-results-for-fourth-quarter-and-full-year-2025)

- **美敦力 Medtronic**：历史上通过大型并购成为全球器械巨头，但近几年增长更多靠现有产品组合修复和创新迭代，2025财年有机收入增长指引约4.75%-5%。来源：[Medtronic FY2025 update](https://news.medtronic.com/2025-02-18-Medtronic-reports-third-quarter-fiscal-2025-financial-results)

- **史赛克 Stryker**：骨科、神经科技、医疗外科设备领域都大量依赖并购扩品类，同时也保持不错的内生增长。它属于“高频小中型并购 + 强渠道整合”的代表。

行业规律大概是：

| 公司阶段       | 增长方式                         |
| -------------- | -------------------------------- |
| 创新型小公司   | 主要靠单品或少数产品内生增长     |
| 中型专科公司   | 内生增长为主，辅以小并购         |
| 全球龙头       | 并购非常重要，常常是核心增长工具 |
| 成熟低增长巨头 | 并购、剥离、重组变成组合管理工具 |

所以回到迈瑞医疗：不能因为它收购 HyTest、惠泰医疗，就简单说增长质量低。更合理的比较标准是看它像不像全球优秀器械龙头那样，把并购资产转化成平台能力。

判断并购质量高低，主要看四个指标：

1. 并购标的是否和主业有协同；
2. 并购后是否还能保持标的自身增长；
3. 是否能通过迈瑞渠道放大收入，尤其海外；
4. 是否没有明显商誉减值、利润虚胖和现金流恶化。

如果这些做到了，外购不是问题，反而是全球医疗器械龙头的标准打法。真正的问题不是“有没有并购”，而是“并购后有没有产生有机增长”。



## 整理历史上top5医疗器械巨头的发展史

下面按**2025年全球医疗器械收入排名**来整理 Top 5：Medtronic、Johnson & Johnson MedTech、Medline、Siemens Healthineers、Stryker。这个口径来自 Medical Design & Outsourcing / MassDevice 的 Medtech Big 100，2025榜单收入分别约为：Medtronic 335亿美元、J&J MedTech 319亿美元、Medline 255亿美元、Siemens Healthineers 242亿美元、Stryker 226亿美元。  
来源：[Medtech Big 100 2025](https://www.medicaldesignandoutsourcing.com/2025-medtech-big-100-worlds-largest-medical-device-companies/)、[MassDevice Top 10](https://www.massdevice.com/largest-medical-device-companies-2025/)

**先给结论**

全球医疗器械巨头的发展史，几乎都不是“单点产品内生长大”的故事，而是：

**技术起家 → 做成核心平台 → 全球渠道扩张 → 并购补赛道 → 剥离低效资产 → 聚焦高增长细分领域。**

迈瑞医疗现在走的“自研 + 并购 + 全球化”路径，其实非常像这些巨头的中后期阶段。

---

**1. Medtronic：从心脏起搏器到全球最大纯器械平台**

Medtronic 1949年成立，早期是医疗设备维修公司。真正改变命运的是心脏起搏器。1950年代，公司联合医生开发可穿戴式电池供电心脏起搏器，奠定心血管电生理和植入器械基因。

之后 Medtronic 从心脏节律管理扩展到脊柱、神经调控、糖尿病、外科设备等领域。1980年代进入神经刺激，2016年推出首个 FDA 批准的混合闭环胰岛素系统。

最关键的外延动作是2015年收购 Covidien。这笔交易让 Medtronic 获得外科、呼吸、监护、耗材等业务，一跃成为全球最大医疗器械公司之一。

发展主线：**心脏起搏器单点突破 → 植入器械平台 → 多治疗领域扩张 → Covidien并购变成综合巨头。**  
来源：[Medtronic 历史](https://www.medtronic.com/en-us/our-company/history.html)

---

**2. Johnson & Johnson MedTech：强生的器械帝国，靠品牌和并购拼出来**

J&J 本身1886年成立，最早不是器械公司，而是无菌敷料、手术用品和消费医疗起家。它的器械业务后来由多个平台拼成：Ethicon 外科缝线和手术器械、DePuy Synthes 骨科、Biosense Webster 电生理、Acuvue 眼科、Abiomed 心衰泵等。

关键节点：

- 1947年收购缝线公司 Merson，后来发展为 Ethicon；
- 1998年收购 DePuy，进入骨科；
- 2012年以约197亿美元收购 Synthes，整合成 DePuy Synthes，成为全球骨科巨头；
- 2022年宣布以约166亿美元收购 Abiomed，强化心血管和心衰治疗；
- 2025年宣布计划分拆骨科业务 DePuy Synthes，进一步聚焦高增长的心血管、外科和眼科。

J&J MedTech 的历史很典型：它不是靠一个器械技术从小长大，而是靠强生集团的资金、品牌、全球渠道和连续并购做成组合平台。

发展主线：**外科耗材起家 → 骨科并购做大 → 心血管和电生理加码 → 剥离低增长骨科，转向高增长器械。**  
来源：[J&J完成Synthes收购](https://www.jnj.com/media-center/press-releases/johnson-johnson-announces-completion-of-synthes-acquisition)、[Abiomed收购公告](https://www.abiomed.com/en-us/about-us/news-and-media/press-releases/johnson-johnson-to-acquire-abiomed)

---

**3. Medline：不是高科技器械巨头，而是医疗供应链巨头**

Medline 和前几家公司不太一样。它更像“医疗耗材 + 医疗供应链 + 私有品牌”的巨头，而不是心脏、影像、骨科这种高技术器械公司。

Medline 1966年由 Mills 家族创立，早期做医疗纺织品和基础耗材。后来进入塑料用品、手术包、长期护理用品、医院供应链服务。1996年进入 prime vendor distribution，也就是医院综合供应商模式。

它的核心能力不是某个独家高端器械，而是：

- 海量 SKU；
- 自有品牌耗材；
- 医院和长期护理机构渠道；
- 采购和物流系统；
- 成本控制与供应链效率。

2021年，Blackstone、Carlyle、Hellman & Friedman 等私募机构参与收购 Medline。此后它继续做大医疗耗材和供应链体系。

发展主线：**医疗纺织品 → 基础耗材 → 手术包和护理用品 → 医院供应链平台。**  
来源：[Medline 历史](https://www.medline.com/about-us/history/)

---

**4. Siemens Healthineers：从X光管到影像、诊断和肿瘤治疗**

Siemens Healthineers 是德国西门子医疗业务发展而来。它的根在影像设备。1896年，西门子制造出用于医学诊断的工业化 X 光管，之后长期在 X 光、CT、MRI、超声、实验室诊断、介入影像等领域积累。

这家公司和 Medtronic、Stryker 不同，它不是植入器械公司，而是“医院大设备 + 诊断系统 + 数字化”的代表。它的壁垒在工程、物理、软件、医院装机基础和服务体系。

关键外延动作是2021年完成收购 Varian。Varian 是全球放疗设备龙头，这让 Siemens Healthineers 从“诊断影像”延伸到“肿瘤治疗”，形成影像诊断、介入治疗、IVD、放疗的组合。

发展主线：**X光和影像设备 → CT/MRI/超声/IVD → 数字化医疗 → 收购Varian进入肿瘤治疗闭环。**  
来源：[Siemens Healthineers 历史](https://www.siemens-healthineers.com/en-us/about-us/history-of-innovation)、[收购Varian公告](https://www.siemens-healthineers.com/press/releases/varian-closing)

**5. Stryker：骨科医生发明起家，靠骨科和外科并购做大**

Stryker 1941年由骨科医生 Homer Stryker 创立。早期产品来自医生对临床痛点的发明，比如病床翻身架、石膏切割器等。它的底色很像“医生创新 + 工程产品化”。

后来 Stryker 从骨科设备扩展到关节、创伤、脊柱、神经科技、手术设备和医院设备。1979年上市后，公司持续通过并购扩品类。

重要并购包括：

- 2013年以约16.5亿美元收购 MAKO Surgical，进入骨科手术机器人；
- 2020年完成收购 Wright Medical，强化肩肘腕手、足踝等创伤和四肢骨科。

Stryker 的厉害之处是并购后执行能力强，尤其 MAKO 机器人和骨科植入物结合，成为它区别于传统骨科公司的关键资产。

发展主线：**骨科医生发明 → 骨科器械 → 关节和创伤平台 → 手术机器人和神经科技 → 高频并购整合。**  
来源：[Stryker 历史](https://www.stryker.com/qa/en/about/history.html)、[收购MAKO公告](https://investors.stryker.com/press-releases/news-details/2013/Stryker-Announces-Definitive-Agreement-to-Acquire-MAKO-Surgical-Corp-for-165-Billion/default.aspx)、[收购Wright Medical公告](https://investors.stryker.com/press-releases/news-details/2020/Stryker-completes-acquisition-of-Wright-Medical/default.aspx)

**共同规律**

这5家公司有几个共同点：

1. **起家通常靠一个强技术或强渠道切口**  
   Medtronic 是起搏器，Siemens 是影像，Stryker 是骨科器械，J&J 是外科耗材，Medline 是医疗供应链。

2. **做大以后都靠并购扩张**  
   Covidien、Synthes、Abiomed、Varian、MAKO、Wright Medical，都是决定性并购。

3. **并购不是低质量增长的代名词**  
   全球器械龙头的常规打法就是买入优质技术平台，再用自己的渠道、注册、临床、制造和客户关系放大。

4. **成熟后会主动剥离低增长资产**  
   J&J 分拆消费健康，又计划分离骨科；Medtronic 也持续调整组合。巨头不是越大越好，而是不断把资源转向高增长、高壁垒领域。

5. **最终竞争力不是单个产品，而是平台能力**  
   包括医生教育、全球注册、医院渠道、装机基础、耗材复购、研发体系、并购整合能力。

所以看迈瑞医疗，可以对标的不是“它有没有并购”，而是它能不能像这些巨头一样，把并购资产转化成自己的长期平台能力。
